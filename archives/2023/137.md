常见服务端口

| 端口 | 用途 |
| --- | --- |
| 21 | FTP 文件传输服务 |
| 22 | SSH 远程连接服务 |
| 23 | TELENT 终端仿真服务 |
| 25 | SMTP 简单邮件传输服务 |
| 53 | DNS 域名解析服务 |
| 80 | HTTP 超文本传输服务 |
| 443 | HTTPS 加密超文本传输服务 |
| 8080 | TCP 服务端默认端口 |
| 8888 | Nginx 服务器端口 |
