## 自定义组件选项配置

诸如 styleIsolation、virtualHost 等自定义组件[配置](https://uniapp.dcloud.net.cn/tutorial/vue3-api.html#%E5%85%B6%E4%BB%96%E9%85%8D%E7%BD%AE)，声明方式如下：

▼ Composition API

若是 Vue3.3+，可使用 [defineOptions()](https://vuejs.org/api/sfc-script-setup.html#defineoptions)。

```html
<script setup>
  defineOptions({
    virtualHost: false,
    styleIsolation: 'shared',
  })

  // your code...
</script>
```

若是 Vue3.3 以下，需要两个 `<script>` 块。

```html
<script setup>
  // your code...
</script>

<script>
  export default {
    options: {
      virtualHost: false,
      styleIsolation: 'shared',
    },
  }
</script>
```

▼ Options API

```html
<script>
  export default {
    options: {
      virtualHost: false,
      styleIsolation: 'shared',
    },
  }
</script>
```
